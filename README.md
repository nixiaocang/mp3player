# mp3player
一个基于SDL2和libmad的简单跨平台mp3播放器， win10 和 ubuntu 16.04 测试通过
## 用到的库
- SDL2            绘图和播放PCM音频
- libmad          解码mp3
- fftw3           计算音频波形
